About

NoteBLOX is an on-page support and collaboration tool for websites.

REST API

See the REST API documentation.

Howto

Prerequisites

  • Java Development Kit 1.8 (either OpenJDK or Oracle)
  • Apache Maven 3.3.9

Checkout

{% highlight bash %} git clone https://github.com/Noteblox/Noteblox-Server.git {% endhighlight %}

Build

{% highlight bash %} mvn clean install {% endhighlight %}

Run

{% highlight bash %} mvn spring-boot:run {% endhighlight %}

Run Integration Tests

{% highlight bash %} mvn clean install -P ci {% endhighlight %}
